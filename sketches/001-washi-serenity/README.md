## Variant: 静寂 · 和紙 (Serenity · Washi Paper)

### Design stance
日式温润极简。将「阅读痕迹」作为空间的核心温度，而非效率指标。像在纸质书上留下铅笔划线和折角——安静、私密、有呼吸感。

### Key choices
- Layout: 单栏列表，每本书一条，宽松间距（48px 封面 + 14px gap）
- Typography: 系统中文字体，标题 15px/600wt 有足够重量感，辅助文字退到背景色
- Color: 暖白纸色 #faf6f0 底色，#c75b4a（朱红）唯一强调色——只用于进度条和痕迹点
- Interaction: 底部标签切换，标签激活态只用一个小圆点标记，不喧哗

### Trade-offs
- Strong at: 安静感、阅读痕迹的可感知性、「精神空间」的氛围
- Weak at: 信息密度低，书多时需要较长滚动；对纯工具型用户可能节奏偏慢

### Best for
- 把阅读 App 当作「精神居所」而非「信息处理工具」的用户
- 享受缓慢、沉浸式浏览体验的人
