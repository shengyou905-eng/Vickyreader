## Variant: 朝日 · 明晰 (Morning Light · Clarity)

### Design stance
现代明亮清爽。把阅读管理做得干净利落，但不失温度。用卡片、进度环、标签系统让信息一目了然，像一个整理得很好的书房——每样东西都在该在的地方。

### Key choices
- Layout: 圆角卡片行，1px 暖灰边框，hover 有微阴影反馈
- Typography: 系统中文字体，标题 15px/600wt，辅助色 #888~#bbb 清晰层级
- Color: 近白底色 #fcfcf9，#e69a3a（琥珀金）为唯一暖色强调——进度环、痕迹标签、小U引用
- Interaction: 底部标签带 emoji 图标 + 文字标签，激活态有填充背景；随心记有浮动 FAB

### Trade-offs
- Strong at: 信息清晰、功能可达性好、现代感、「有条理的温度」
- Weak at: 边框和卡片结构让屏幕比方案 A「碎」一些；emoji 图标风格可能不够统一

### Best for
- 需要清晰管理多本书和阅读进度的用户
- 喜欢结构化、干净界面的读者
